## General info
This is one page RESTful API. using expressJS and mongoDB.
	
<br>

## Technologies
Project is created with:
* EJS version 3.1.6
* Express version 4.17.1
* Mongoose version 5.12.13

<br>

## Setup
To run this project, install it locally using npm:

```
$ cd to the project /Wiki-API
$ npm install
$ node app.js
```
